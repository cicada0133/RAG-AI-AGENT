# Central AI — AI-платформа глубокой переработки знаний

> Альтернатива Google NotebookLM с локальным open-source LLM

---

## Что сделано

**Стек:**
- **Frontend:** Next.js — три вкладки: Chat, Mindmap, Podcast
- **Backend:** FastAPI + LangChain + ChromaDB (RAG-пайплайн)
- **LLM:** Ollama + `qwen2.5:7b` — полностью локально, без внешних API
- **TTS:** `edge-tts` — озвучка на русском
- **Деплой:** Docker Compose, запуск одной командой

---

## Реализованный функционал

| Фича | Статус |
|---|---|
| Загрузка документов (PDF, DOCX, TXT) | ✅ Работает |
| RAG-чат с источниками | ✅ Работает |
| Mindmap / интеллект-карта | ✅ Работает |
| Аудиопересказ (подкаст) | ✅ Работает |

---

## Ключевое архитектурное решение

**Нет зависимости от зарубежных API** — закрывает главный критерий оценки:
- `qwen2.5:7b` работает полностью на локальном GPU
- Данные пользователя не покидают машину
- Смена модели — одна переменная в конфиге

---

## Запуск

```bash
docker compose up -d
docker exec central_invest-ollama-1 ollama pull qwen2.5:7b
docker compose restart backend
# Открыть http://localhost:3050
```
